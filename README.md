# AI Agent Hub (FINAL THEME BUILD)

Статичный сайт с 3 темами, переключением языка и валюты, каталогом агентов (аккордеон), страницами поддержки и предложений.

## Быстрый старт
Откройте `index.html` в браузере. Все страницы статичны, бэкенд не требуется.

## Темы
- Оранжево‑чёрная (по умолчанию)
- Тёмно‑синяя
- Светлая (аккуратная, с мягким градиентом)

Тема хранится в `localStorage` (ключ `aiah_theme`). Макет макбука на странице `about.html` автоматически меняется в зависимости от выбранной темы.

## Настройки
Страница `settings.html` позволяет выбрать тему, язык (`aiah_lang`) и валюту (`aiah_currency`). Есть кнопка сброса настроек.

## Каталог
`catalog.html` — карточки агентов в виде аккордеона (открыта только одна). Управление — в `assets/js/app.js`.

## Регистрация
На главной (`index.html`) есть кнопка «Регистрация», открывается модальное окно (`<dialog>`), функционал не реализован.
